What is proximate?

Proximate is a framework/middleware that empowers proximity communication applications using peer-to-peer technology. Designed for mobile devices with wireless connectivity, this platform can host pluggable applications sharing the same network and user identification layers. It can be compared to MSN/Windows Live Messenger in the way it integrates diverse modules in a common environment, allowing people to communicate and interact.

Use cases

Here is a short list of scenarii justifying the use of proximate:

  • in a classroom, student can share notes and comments in real-time
  • make new friends by chatting with randoms around you

Status of the current release

This release demonstrates the idea and the technology of proximate. It is unstable but happened to work! You can install it on your PocketPC (Windows Mobile 2003 with dotNet framework 1) and try the sample chat application.

Features of the chat application

Here is a list of features proposed in the chat application:

  • scribble to your mates using the stylus
  • exchange text messages
  • transmit (light-weighted) files
  • chat sessions are private but you can invite other mates to join
  • switch between chat sessions you're involved with
  • mates don't have to be "added" in the system, anyone running proximate around is visible
  • user profiles with picture and user status (online, busy, free for chat...)

Strengths of proximate

  • no server is needed, peer-to-peer is magic!
  • anybody can develop new applications using the existing network layer
  • it is free for personnal use

How to run it?

  1. You need to have a Microsoft dotNet compact framework installed on your PocketPC.
  2. Download and run the CAB file corresponding to your configuration on your PocketPC.
  3. Save your opened files and backup any valuable data of your PocketPC. Again, proximate is unstable and may crash your system.
  4. Connect to a wireless network and run proximate. You can find it in "Program Files", using the explorer.

The project and its authors

Proximate was developed as a "Mobile Devices Project" at Queensland University of Technology (QUT), Brisbane, Australia. Sanjot Dalvi and Adrien Joly spent 4 months on it and eventually got the excellent grade of 7/7.
